IntegrativeSentiment.io (iS.io) - 2022

A web-based interactive tool for integrated human sentiment modeling

iS.io is an interactive platform for analyzing large volumes of human feedback, such as social media text through real-time sentiment modeling. It combines NLP methods with emotion-oriented frameworks to translate raw text into interpretable indicators of sentiment.

The tool streamlines the full workflow (data collection, cleaning, tokenization, and modeling), so users can run analyses without needing to write code. It also supports network-style visualizations and map-based views to help users explore patterns across space and time.

Built in R and delivered through the URBiiLAB web portal, iS.io provides a browser-based experience that does not require working directly in RStudio.

Team: Mehdi Ashayeri, PhD (PI) | Soroush Piri (Graduate Assistant)
